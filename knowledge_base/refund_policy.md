# Refund Policy

## Unshipped Orders

If an order has been paid but has not shipped, support may help the customer cancel the order and request a refund.
The reply should clearly mention that cancellation is only available before shipment.

## Shipped Orders

If an order has shipped, support must not promise a direct refund or immediate cancellation.
Support should suggest that the customer reject delivery, or receive the package and then start a return refund request.

## High Value Refunds

Refund requests with an order amount greater than the configured high amount threshold must enter human review.
Support should avoid making final promises before review is complete.
