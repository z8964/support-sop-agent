# Logistics Policy

## No Tracking Update

If tracking has not updated for more than two days, support should explain the current logistics state and create a follow-up if needed.
The reply should not claim that the package is lost unless the carrier status confirms it.

## In Transit

If the package is still in transit and the delivery promise has not expired, support should explain that it is moving through the carrier network.
Support may ask the customer to wait for the next carrier update.

## Lost Package

If the carrier confirms a package is lost, support should create an escalation ticket and move the customer case to human review.
